# ⚡ Mashkal Overlay v2.0

Advanced iOS Overlay System with ImGui + Metal

## Features

- 🎨 **Modern ImGui Interface** - Professional dark theme with blue accents
- 🎮 **Metal Rendering** - High-performance GPU-accelerated graphics
- 🔐 **Security System** - 7-day activation with password protection
- 📱 **Touch Support** - Full touch input handling
- 🎯 **Interactive Menu** - 5 tabs with multiple features
- ⚡ **Performance** - 60 FPS rendering with FPS counter
- 🎛️ **Customization** - Adjustable opacity, animations, colors

## Installation

1. Install via Cydia/Sileo/Zebra
2. Respring device
3. Activate with password: `halak`

## Usage

- Tap the floating ⚡ button to toggle overlay
- Drag the header to move the window
- Pinch to resize
- Use tabs to navigate features

## Menu Tabs

1. **الرئيسية** - Dashboard with stats and quick actions
2. **الخصائص** - Feature toggles and settings
3. **الإعدادات** - Appearance and advanced options
4. **الحماية** - Activation status and management
5. **معلومات** - About and device info

## Technical Details

- Target: iOS 14.0+
- Architectures: arm64, arm64e
- Frameworks: UIKit, Metal, MetalKit, Security
- Dependencies: mobilesubstrate
